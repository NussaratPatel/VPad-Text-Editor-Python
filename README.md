# VPad Text Editor

## Overview

VPad Text Editor is a desktop text editing application developed using Python and Tkinter. It provides essential text editing features along with advanced formatting and customization options.

## Technologies Used

* Python
* Tkinter GUI Library

## Features

### File Operations

* New File
* Open File
* Save File
* Save As
* Exit with Save Confirmation

### Text Editing

* Copy
* Cut
* Paste
* Clear All
* Find and Replace

### Text Formatting

* Font Family Selection
* Font Size Selection
* Bold Text
* Italic Text
* Underline Text
* Font Color Change

### Text Alignment

* Left Alignment
* Center Alignment
* Right Alignment

### User Interface Features

* Toolbar
* Status Bar
* Multiple Color Themes
* Keyboard Shortcuts
* Scrollbar Support

## Keyboard Shortcuts

| Action    | Shortcut       |
| --------- | -------------- |
| New File  | Ctrl + N       |
| Open File | Ctrl + O       |
| Save File | Ctrl + S       |
| Save As   | Ctrl + Alt + S |
| Find      | Ctrl + F       |
